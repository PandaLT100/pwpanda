What is JavaScript?

JavaScript is a language built to make your website function. JavaScript can calculate data, manipulate it, and even validate it. It CANNOT store data. That function is something for backend development. Yes, JavaScript can work with both frontend and backend development, BUT we are focusing on frontend development.

One thing to note about JS (JavaScript) is that it is a language that works in order from top to bottom.

Example: Let's look at the list of instructions below.

Hello, JavaScript. Today, I want you make cereal in this specific order.

  1. Go to the kitchen
  2. Grab the cereal box
  3. Grab the milk from the fridge
  4. Grab a bowl and a spoon from the counter
  5. Pour the cereal into the bowl
  6. Pour the milk into the bowl
  7. Eat!

Once you have given JavaScript the command, it will do what you say in the order you tell it to do it.

There IS a something you have to watch out for, though. Let's say you reordered the list so that JavaScript grabs the bowl AFTER pouring the cereal. Technically, there is no bowl to pour it into. So, JavaScript will give you an error saying that it cannot eat something that is on the floor. So, when writing in JavaScript, you must be careful with these small technicalities.
